协议支持基本类型如下
boolean int long double bytes

bytes对应java 的bytes[]

boolean[] int[] long[] double[] 这些数组对应 java的 boolean[] int[] long[] double[]

对于对象的支持，支持使用消息对象，如果不是一个包的那么需要写全部限定名
对象也可以有数组如 Abc[],对应java 的ArrayLiat<Abc>而不是Abc[]


生成消息字段默认全部是null 和 0 false 这些初始值
